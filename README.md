# My Shell Configuration

A personal collection of shell configurations and customizations to enhance terminal productivity and aesthetics.

## Overview

This repository contains my personalized shell configurations, including environment settings, aliases, functions, and custom prompts. It serves as a centralized location to maintain and version control all shell-related customizations across different machines and environments.

## Contents

This repository will include:

- **Shell Configurations**: Custom `.bashrc`, `.zshrc`, and other shell-specific configuration files
- **Custom Prompts**: Beautiful and informative command-line prompts with git status, directory info, and more
- **Aliases**: Convenient shortcuts for common commands
- **Functions**: Useful shell functions for various tasks
- **Environment Variables**: Standard environment setup for development tools

*Note: Shell configurations and prompt files are being added progressively.*
